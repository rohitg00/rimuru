# Keyboard Shortcuts

## TUI Keybindings

### Tab Navigation

| Key | Tab |
|-----|-----|
| `1` | Dashboard |
| `2` | Agents |
| `3` | Sessions |
| `4` | Costs |
| `5` | Models |
| `6` | Metrics |
| `7` | Plugins |
| `8` | Hooks |
| `9` | MCP |
| `0` | Help |

### Navigation

| Key | Action |
|-----|--------|
| `j` / Down | Move down |
| `k` / Up | Move up |
| `Tab` | Next tab |
| `Shift+Tab` | Previous tab |
| `Enter` | Select / expand |

### Actions

| Key | Action |
|-----|--------|
| `r` | Refresh current tab data |
| `s` | Sync agents |
| `t` | Cycle through themes |
| `q` | Quit |
| `Ctrl+c` | Force quit |

### Theme Cycling

Press `t` to cycle through 15 themes:

1. Rimuru Slime
2. Great Sage
3. Predator
4. Veldora
5. Shion
6. Milim
7. Diablo
8. Tokyo Night
9. Catppuccin
10. Dracula
11. Nord
12. Gruvbox
13. Rose Pine
14. Cyberpunk
15. Tempest

You can also start with a specific theme:

```bash
rimuru-tui --theme "Rimuru Slime"
```
